[Github+Hexo]搭建自己的独立博客

前言：刚开始尝试用Python做项目，遇到问题常常穿梭于各大博客寻求解决，继而萌生了自己写博客的想法。我们可以在博客分享做项目遇到的坑，也可以对自己学到的知识进行梳理。作为一个爱学(zhuang)习(B)的程序员，我选择了在Github上搭建自己的个人博客，当参考着别人的教程，成功地把博客搭建起来了，第一篇博文也就有素材了。

一、前期准备

  1. 到GitHub官网上注册自己的账号，用户名还是要好好取的，毕竟跟博客域名有关。
  2. 安装Git，直接到Git官网下载安装就行。（身为程序员，Git大概还是很有用的）
  3. 安装node.js环境，可以到node.js官网下载自己系统对应的版本进行安装。

二、建立仓库

登陆GitHub，在头像隔壁的加号点击 New repository，新建一个代码仓库。

进入代码仓库创建页面，Repository name填 “username.github.io”，其中username是自己的名字，最好跟注册GitHub时的用户名一致，当然，不一样也可以，受影响的只是访问博客的地址。

填写好仓库名后，点击下方的 Create repository 创建代码仓库。
创建成功之后，点击 Setting。

其中 Rename 是更改仓库名，说不定你会用到。
拉到下面，找到 GitHub Pages，访问博客的网址在这里。
